## GitHub Account Types
### Type I - Personal Account
  - Identity on GitHub.com (username & profile)
  - Owns repos, packages, projects
  - Actions attributed to the user
  - Plans available: Free, Pro
  - Unlimited public & private repos (private features limited on Free)

### Type II - Organization Account
  - Shared account for collaboration
  - Tiered permissions & roles
  - Members act via personal accounts
  - Owners & security managers manage settings
  - Plans available: Free, Team

### Type III - Enterprise Account
  - Centralized policy & billing across organizations
  - Enables inner sourcing between orgs
  - Has a handle (like user/org)
  - Enterprise owners manage/transfer/create orgs
  - Enforce org-level policies via enterprise policies
  - Plan: Enterprise


## GitHub Plans
### Type I - GitHub Free for personal accounts and organizations
  - Personal: unlimited repos, limited private features
  - Organization: unlimited repos, limited private features
### Type II - GitHub Pro for personal accounts
  - Advanced tools for individuals
  - More Actions minutes & Codespaces
### Type III - GitHub Team
  - For organizations
  - Enhanced collaboration & security
### Type IV - GitHub Enterprise
  - For enterprises
  - Advanced security, compliance, deployment
